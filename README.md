# Library Management System





This was made for Sir Jamie Eduardo Rosal, as my submission for his OOP Subject. 





This Java Swing application utilizes JDBC and an SQL database. This application shows all functions, from adding books, seeing books, editing books, and deleting books. 

This application also features a search function, where the user can search through the book's title, author, or ISBN. 




# DB.properties
